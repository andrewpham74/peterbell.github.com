Getting started with nodejs on OSX Lion

1. Install Homebrew/node

If you don't have Homebrew yet, it's time. It is a package manager that makes it easy to download and update packages on a Mac just seems to work better for installing the occasional package and to be more popular than MacPorts or Fink. Check out the installation instructions.

Once you've installed homebrew, you just need to update it to ensure you get the latest version of the packages you want to download. Then it's a one line command to install node. At a command line, type:

brew update
brew install node

If you really don't want to install brew, there is an installer or you can build from source. More information is available on the node website.

Test your node installation by checking the version of node at the command line:

node -v

As long as it returns a version number and it is has an even number for the first decimal place (node uses semantic versioning with odd versions like 0.5 and 0.7 being unstable and 0.4 and 0.6 being stable production releases) you're good to go.

2. Installing NPM

To do anything useful with node, you're going to want to download the Node Package Manager. NPM allows you to easily download an install the packages you need to manage. It's an indispensable tool for working with node.

To install, in a terminal window just type:

curl http://npmjs.org/install.sh | sh

Then you'll need to add /usr/local/lib/node_modules to your path. Personally I set my path in my ~/.bash_profile. Once you've added the path, open a new terminal and check that your path contains the new directory:

echo $PATH

And that's it - you're ready to roll!
